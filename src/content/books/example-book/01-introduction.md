## 什么是 TypeScript？

TypeScript 是 JavaScript 的超集，添加了可选的静态类型和基于类的面向对象编程。

### 为什么使用 TypeScript？

- **类型安全**：在编译时捕获错误
- **更好的 IDE 支持**：自动补全、重构等
- **更易维护**：代码更加清晰和可预测

### 快速开始

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```

这就是 TypeScript 的基本介绍。下一章我们将学习基础语法。
