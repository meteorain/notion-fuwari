## 关于本书

这是一本示例图书，用于演示博客的图书翻译功能。

### 如何使用

1. 在 `src/content/books/` 下创建新的图书目录
2. 添加 `_meta.md` 文件作为图书元信息
3. 添加章节文件，使用 `01-xxx.md`、`02-xxx.md` 格式命名

### 目录结构示例

```
src/content/books/
├── my-book/
│   ├── _meta.md          # 图书元信息
│   ├── 01-introduction.md # 第1章
│   ├── 02-basics.md       # 第2章
│   └── 03-advanced.md     # 第3章
```
